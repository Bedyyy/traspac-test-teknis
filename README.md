# traspac-test-teknis

Folder "siswai-pns" merupakan pengerjaan terhadap soal Test Kemampuan Teknis sedangkan folder "traspac-penalaran" merupakan pengerjaan terhadap soal Test Kemampuan Penalaran.

## Instalasi
1. Silahkan clone repository ini
2. Pastikan sudah terinstall requirement, yaitu composer, web server (xampp/laragon), python.

### Menggunakan Siswai-PNS
1. Buka folder "siswai-pns" pada text editor anda
2. Import manual database "siswa-pns.sql" pada folder database ke dalam local database anda
3. Buka terminal dan ketikkan perintah "composer update"
4. Selanjutnya ketikkan perintah "php artisan serve" pada terminal
5. Aplikasi siap dijalankan

### Menggunakan Program Python
1. Buka folder "traspac-penalaran" pada text-editor anda
2. Jalankan file main.py
3. Akan terbuka terminal dengan menu aplikasinya
4. Silahkan masukkan input yang diminta pada terminal
